## Quick Start

```bash
# Install dependencies
cd front
npm install

cd back
npm install

# Start Express Server: http://localhost:4000
cd back
node app.js

# Start Vue DevServer: http://localhost:8080
cd front
npm run serve

# Build for production (Will build into server/public, ready for deployment)
cd client
npm run build
```
